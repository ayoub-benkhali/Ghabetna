# Catégories d'incidents forestiers

## Incendie (feu)
Un incendie forestier est une urgence absolue.
Signes à observer : fumée visible, flammes, odeur de brûlé, arbres carbonisés.
Que faire : Signalez immédiatement sur Ghabetna ET appelez le 1800 (Protection Civile).
Cochez toujours "Incident critique" pour un incendie.
Description recommandée : localisation précise, étendue approximative, direction du vent, présence de personnes en danger.

## Coupe illégale (coupe_illegale)
Abattage non autorisé d'arbres dans une zone protégée.
Signes à observer : souches fraîches, troncs coupés, sciure au sol, bruit de tronçonneuse, camions suspects chargés de bois.
Que faire : Signalez sur Ghabetna, notez les plaques d'immatriculation si possible. Ne confrontez pas les personnes.
Description recommandée : nombre d'arbres coupés, type d'arbres, présence de véhicules, heure d'observation.

## Maladie végétale (maladie)
Infection ou parasite affectant les arbres ou plantes de la forêt.
Signes à observer : feuilles jaunies ou brunes anormalement, champignons sur les troncs, insectes en masse sur les arbres, arbres morts groupés.
Que faire : Signalez sur Ghabetna avec des photos si possible. Ne touchez pas les plantes malades.
Description recommandée : type de végétation affectée, étendue de la zone, symptômes observés.

## Déchets (dechet)
Dépôt illégal de déchets en forêt.
Signes à observer : sacs poubelles abandonnés, déchets industriels, déversement de liquides suspects, décharges sauvages.
Que faire : Signalez sur Ghabetna. Ne touchez pas les déchets chimiques ou industriels.
Description recommandée : type de déchets, quantité approximative, accessibilité de la zone.

## Trafic (trafic)
Activités illégales de commerce ou transport en forêt.
Signes à observer : véhicules suspects circulant la nuit, personnes transportant des animaux sauvages, vente illégale de plantes protégées.
Que faire : Signalez sur Ghabetna discrètement. Ne vous approchez pas des personnes suspectes.
Description recommandée : type de véhicule, nombre de personnes, nature des marchandises si visible.

## Refuge suspect (refuge_suspect)
Construction illégale ou occupation non autorisée en forêt.
Signes à observer : cabanes illégales, campements permanents non autorisés, installations électriques sauvages.
Que faire : Signalez sur Ghabetna avec photos si possible. Ne pénétrez pas dans les structures.
Description recommandée : type de construction, taille approximative, présence de personnes.

## Autre
Pour tout incident forestier ne rentrant pas dans les catégories précédentes.
Exemples : braconnage, pollution sonore excessive, dégradation de panneaux de signalisation, occupation illégale de chemins forestiers.
Description recommandée : décrivez le plus précisément possible ce que vous avez observé.
