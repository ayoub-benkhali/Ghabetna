# Guide d'utilisation de Ghabetna

## Qu'est-ce que Ghabetna ?
Ghabetna est une plateforme intelligente tunisienne de surveillance et de signalement des incidents forestiers. Elle permet aux citoyens, visiteurs et professionnels de signaler et suivre les incidents dans les forêts tunisiennes.

## Comment créer un compte visiteur ?
1. Ouvrez l'application Ghabetna
2. Sur la page de connexion, cliquez sur "Pas encore de compte ? Créer un compte visiteur"
3. Remplissez le formulaire : nom complet, email, téléphone (optionnel), CIN (optionnel), type de visiteur
4. Choisissez votre type : Habitant, Chasseur, Campeur, Randonneur, ou Autre
5. Créez un mot de passe d'au moins 8 caractères
6. Cliquez sur "Créer mon compte"
7. Connectez-vous avec votre email et mot de passe

## Comment signaler un incident ?
1. Connectez-vous à votre compte visiteur
2. Sur l'accueil, cliquez sur "Signaler un incident"
3. Choisissez le type d'incident parmi : Incendie, Coupe illégale, Maladie végétale, Déchets, Trafic, Refuge suspect, Autre
4. Ajoutez une description détaillée de ce que vous avez observé
5. Votre position GPS est détectée automatiquement (vous pouvez la rafraîchir)
6. Ajoutez une photo si possible
7. Cochez "Incident critique" si la situation nécessite une intervention urgente
8. Cliquez sur "Envoyer le signalement"

## Comment voir mes signalements ?
1. Sur l'accueil visiteur, cliquez sur "Mes signalements"
2. Vous verrez la liste de tous vos incidents signalés
3. Chaque incident affiche son statut : En attente, En cours, Résolu, Rejeté
4. Cliquez sur le bouton "+" pour ajouter un nouveau signalement

## Statuts des incidents
- **En attente** : Votre signalement a été reçu et attend d'être traité
- **En cours** : Un agent professionnel traite votre signalement
- **Résolu** : L'incident a été traité avec succès
- **Rejeté** : Le signalement n'a pas pu être confirmé

## Mon profil visiteur

Pour accéder à votre profil :
1. Connectez-vous à votre compte visiteur
2. Sur la page d'accueil, cliquez sur la carte "Mon profil"
3. Vous verrez vos informations : nom complet, email, rôle (Visiteur)

Informations disponibles dans le profil :
- Nom complet
- Adresse email
- Rôle : Visiteur de forêt

Important : La modification du nom ou de l'email n'est pas disponible directement dans l'application. Le profil est en lecture seule.

Pour vous déconnecter :
1. Allez dans "Mon profil"
2. Cliquez sur le bouton rouge "Se déconnecter" en bas de la page
OU
1. Cliquez sur l'icône de déconnexion en haut à droite de l'accueil

## Navigation dans l'application visiteur

L'accueil visiteur contient 4 éléments principaux :
1. Bannière GhabetnaBot (assistant IA) — en haut, fond violet
2. Carte "Signaler un incident" — pour signaler un problème
3. Carte "Mes signalements" — pour voir vos incidents envoyés
4. Carte "Mon profil" — pour voir vos informations et vous déconnecter

Il n'y a pas de menu latéral ni de barre de navigation en bas. Tout se fait depuis l'accueil.
